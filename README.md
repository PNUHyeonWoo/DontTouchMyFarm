- 프로젝트 명 : DontTouchMyFarm

- 목적: 유니티 엔진 및 협업 연습 목적으로 개발한 농사 디펜스 게임입니다.
- 개발 규모: 3인 개발
- 개발 기간: 2023.01 ~ 2023.03

- 홍주혁(ido_013): 몬스터 인공지능 개발, 3D모델링 및 배경 오브젝트 배치
